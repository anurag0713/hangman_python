# Hangman
Made using python.  
words.py contains a list with 1k+ words to guess from. 


## Modes
There are 3 modes:  
Easy - Easy. You get 8 lives.  
Normal - Not very easy, not very hard. You get 5 lives.  
Hell - Literal hell, very hard! You get 3 lives.
